# Lesson 3.1: Programming and PHP Overview

## WHAT IS PHP

- PHP stands for "Hypertext Preprocessor"

- It is a server-side processing language

- Developed by Rasmus Lerdorf in 1994

- PHP is a free language to download and use

- Highly suitable for web application development

## PHP FILE
- PHP files contain text, HTML, CSS, JavaScript, and PHP code.
- PHP files are executed on the server, and the resulting output is returned to the browser as HTML.
- PHP files have the extension .php

### WHY DO WE CHOOSE PHP?
- It is a cross-platform language (runs on Windows, Linux, Mac OS X, etc.).
- It is compatible with almost all servers used today.
- It is a language used by more than 80% of websites worldwide, including google.com, facebook.com, and many others.
